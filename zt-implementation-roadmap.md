# Zero Trust Implementation Roadmap

> 24-month phased implementation plan mapped to CISA Zero Trust Maturity Model stages

See full roadmap documentation in [zt-implementation-roadmap-and-assessment.md](./zt-implementation-roadmap-and-assessment.md).

## Phase Summary

| Phase | Months | Focus | CISA Target |
|---|---|---|---|
| Phase 0 | 1-2 | Foundation: governance, inventory, baseline | Pre-Initial |
| Phase 1 | 1-4 | Identity: MFA, CA, PAM, SSO | Initial - Identity |
| Phase 2 | 3-6 | Devices: MDM, compliance, EDR | Initial - Devices |
| Phase 3 | 5-8 | Network: DNS security, visibility | Initial - Networks |
| Phase 4 | 7-12 | Applications: ZTNA pilot, app controls | Advanced - Apps |
| Phase 5 | 9-15 | Data: classification, DLP | Advanced - Data |
| Phase 6 | 12-18 | Network segmentation | Advanced - Networks |
| Phase 7 | 16-24 | Optimization: phishing-resistant MFA, VPN elimination, automation | Advanced to Optimal |
